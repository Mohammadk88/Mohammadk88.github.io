🚀 Quick Start Guide

5 خطوات من صفر لسيستم شغال

دليل عملي — مو نظري. كل خطوة فيها شو بالضبط لازم تعمل وشو تحتاجه.

🗺️ شوف مخطط النظام أولاً
01

احصل على WhatsApp Business API

الخطوة الوحيدة اللي تحتاج حساب Meta

🔧
Meta for Developers App
developers.facebook.com → Create App → Business → WhatsApp
📱
WhatsApp Business Account
بتحتاج رقم هاتف جديد ما مرتبط بواتساب حالياً
🔑
Phone Number ID + Access Token
من لوحة Meta → WhatsApp → API Setup. احفظهم — بتحتاجهم بالـ n8n
02

جهّز السيرفر

VPS + Docker + n8n + Domain — مرة واحدة وبيشتغل للأبد

🖥️
VPS — Hetzner CX22 أو DigitalOcean
~5$/شهر. Ubuntu 22.04. 2 CPU, 4GB RAM — كافي لـ n8n وPostgreSQL.
🐋
Docker + n8n
docker run -d --name n8n -p 5678:5678 n8nio/n8n
🔒
Domain + SSL
مطلوب للـ Webhook. ~10$/سنة + Certbot مجاناً.
03

أنشئ قاعدة البيانات

PostgreSQL — جدولين كافيين للبداية

🗄️
جدول customers
id, phone (UNIQUE), name, last_order_id, status, created_at
💬
جدول messages
id, phone, text, intent, urgency, needs_human, created_at
🔗
ربط n8n بالـ Database
Credentials → Postgres → host: host.docker.internal
04

استورد الـ Workflows

Workflow 1 + Workflow 2 — Import from JSON

⚡
Workflow 1: Intent Routing
Webhook → Extract → Gemini AI → Decision Router → PostgreSQL → Reply → Log
🔄
Workflow 2: Customer Follow-up
Schedule (يومي 10AM) → استرجاع الزباين → بناء رسالة مخصصة → إرسال
🔑
Variables مطلوبة
GEMINI_API_KEY + PHONE_ID + WHATSAPP_TOKEN — Settings → Variables
05

اختبر وفعّل

3 اختبارات قبل التشغيل الحقيقي

🧪
اختبار 1 — رسالة order_status
"where is my order 4823" — المفروض يجاوب تلقائياً
😠
اختبار 2 — شكوى
"I have a complaint" — المفروض يصعّد لموظف
🗄️
اختبار 3 — تحقق من قاعدة البيانات
SELECT * FROM messages — المفروض تلاقي record جديد
